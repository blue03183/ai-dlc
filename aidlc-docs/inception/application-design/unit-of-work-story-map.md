## 크로스 유닛 스토리

모든 스토리가 백엔드 + 프론트엔드에 걸쳐 있으므로 (장바구니 US-301~303 제외), 통합 테스트 시 백엔드-프론트엔드 간 API 연동 검증이 필수입니다.

| 유형 | 스토리 |
|---|---|
| 백엔드 전용 | 없음 (모든 API는 프론트엔드에서 호출) |
| 프론트엔드 전용 | US-301, US-302, US-303 (장바구니 — 로컬 전용) |
| 크로스 유닛 | 나머지 17개 스토리 |
