# 테이블오더 서비스 — 컴포넌트 정의

---

## 1. 백엔드 (NestJS)

### 1.1 AuthModule
- **목적**: 인증 및 인가 처리
- **책임**:
  - 관리자 로그인/로그아웃 (JWT 발급, 16시간 세션)
  - 테이블 태블릿 인증 (매장ID + 테이블번호 + 비밀번호)
  - JWT 토큰 검증 미들웨어 (Guard)
  - 로그인 시도 제한
- **주요 엔티티**: Admin, TableSession

### 1.2 StoreModule
- **목적**: 매장 정보 관리 (멀티테넌트 기반)
- **책임**:
  - 매장 데이터 관리
  - 매장별 데이터 격리 (멀티테넌트)
- **주요 엔티티**: Store

### 1.3 TableModule
- **목적**: 테이블 및 세션 라이프사이클 관리
- **책임**:
  - 테이블 초기 설정 (번호, 비밀번호)
  - 테이블 세션 시작/종료
  - 이용 완료 처리 (주문 이력 이동, 리셋)
- **주요 엔티티**: Table, TableSession

### 1.4 MenuModule
- **목적**: 메뉴 CRUD 및 카테고리 관리
- **책임**:
  - 메뉴 등록/조회/수정/삭제
  - 카테고리 관리
  - 메뉴 노출 순서 관리
  - 이미지 파일 업로드 처리
- **주요 엔티티**: Menu, Category

### 1.5 OrderModule
- **목적**: 주문 생성, 조회, 상태 관리
- **책임**:
  - 주문 생성 (장바구니 → 주문 변환)
  - 주문 조회 (현재 세션, 과거 이력)
  - 주문 상태 변경 (대기중 → 준비중 → 완료)
  - 주문 삭제 (관리자 직권)
  - 주문 이력 관리 (이용 완료 시 과거 이력 이동)
- **주요 엔티티**: Order, OrderItem, OrderHistory

### 1.6 EventModule
- **목적**: SSE 기반 실시간 이벤트 전송
- **책임**:
  - 관리자 대시보드 SSE 연결 관리
  - 고객 주문 상태 SSE 연결 관리
  - 주문 생성/상태 변경/삭제 이벤트 브로드캐스트
  - 연결 관리 (매장별, 테이블별 구독)

### 1.7 AdminModule
- **목적**: 관리자 계정 관리
- **책임**:
  - 관리자 계정 생성 (오너가 생성)
  - 관리자 목록 조회
  - 역할 관리 (오너/일반 관리자)
- **주요 엔티티**: Admin

### 1.8 UploadModule
- **목적**: 파일 업로드 처리
- **책임**:
  - 이미지 파일 업로드 및 저장
  - 업로드된 파일 URL 반환
  - 파일 삭제

## 2. 고객용 프론트엔드 (React)

### 2.1 AuthPage
- **목적**: 테이블 초기 설정 및 자동 로그인
- **책임**: 매장ID/테이블번호/비밀번호 입력, 로컬 저장, 자동 로그인

### 2.2 MenuPage
- **목적**: 메뉴 조회 및 탐색 (기본 화면)
- **책임**: 카테고리별 메뉴 표시, 메뉴 카드 레이아웃, 상세 정보 모달

### 2.3 CartComponent
- **목적**: 장바구니 관리
- **책임**: 메뉴 추가/삭제/수량 조절, 총 금액 계산, 로컬 스토리지 저장

### 2.4 OrderPage
- **목적**: 주문 확정 및 내역 조회
- **책임**: 주문 확인/확정, 주문 내역 표시, SSE 기반 상태 실시간 업데이트

### 2.5 SSEClient
- **목적**: SSE 연결 관리
- **책임**: 서버 SSE 연결, 이벤트 수신, 자동 재연결

## 3. 관리자용 프론트엔드 (React)

### 3.1 LoginPage
- **목적**: 관리자 로그인
- **책임**: 매장ID/사용자명/비밀번호 입력, JWT 토큰 저장

### 3.2 DashboardPage
- **목적**: 실시간 주문 모니터링 대시보드
- **책임**: 테이블별 그리드 레이아웃, 주문 카드 표시, 신규 주문 강조, 필터링

### 3.3 OrderDetailModal
- **목적**: 테이블별 주문 상세 보기
- **책임**: 전체 주문 목록, 상태 변경, 주문 삭제

### 3.4 TableManagementPage
- **목적**: 테이블 관리
- **책임**: 테이블 설정, 이용 완료 처리, 과거 내역 조회

### 3.5 MenuManagementPage
- **목적**: 메뉴 관리
- **책임**: 메뉴 CRUD, 이미지 업로드, 순서 조정

### 3.6 AdminManagementPage
- **목적**: 관리자 계정 관리 (오너 전용)
- **책임**: 관리자 계정 생성, 목록 조회

### 3.7 SSEClient
- **목적**: SSE 연결 관리
- **책임**: 서버 SSE 연결, 주문 이벤트 수신, 자동 재연결
